## Features
-[x] File Manage
-[x] Folder Manage
-[x] File & Folder Share
-[x] Storage Capacity Control
-[ ] RecycleBin
-[x] Support AliOSS
-[ ] Support Upyun
-[ ] Support Qiniuyun
