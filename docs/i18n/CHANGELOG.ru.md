# Журнал изменений

## v2.7.3 — 2026-06-09

### Возможности
- **Страница «О программе»** — новая страница в админ-панели с информацией об
  инстансе, редакции и версии, со встроенной панелью журнала изменений и
  проверкой последней версии по GitHub Releases.
- **Коммерческое лицензирование** — поддержка отдельной коммерческой
  авторизации, лента редакции в макете админки и сравнительная таблица
  редакций, сгруппированная по возможностям (с гейтингом соцлогина и
  загрузчика).
- **Управление правами** — администраторы теперь могут редактировать и
  отзывать выданные права на квоту.
- **Телеметрия инстанса** — необязательная анонимная отправка сведений о
  развёртывании (с регионом по GeoIP), чтобы понимать, как используется ZPan.

### Исправления
- Более устойчивый учёт расхода при удалённой загрузке.
- Исправлен запуск Docker-образа (теперь под защитой CI), для регистрации
  загрузчика используется имя хоста.
- Ускорен список квот в админке — пакетные запросы (разбитые с учётом лимита
  D1 в 100 параметров), ежемесячный сброс перенесён в задачу по расписанию.
- Версия приложения теперь определяется из `package.json` и внедряется на
  этапе сборки.

## v2.7.0 — 2026-06-06 · Удалённые загрузки, WebDAV и многое другое

### Возможности
- **Менеджер удалённых загрузок** — перенаправляйте torrent/HTTP-загрузки на удалённые воркеры
  с подробным инспектором задач, гео-регионами пиров, удержанием BT-сидов и
  загрузкой обратно на ваш диск с сохранением структуры папок.
- **CLI-загрузчик `zpan`** — вход на устройстве одной командой и настраиваемый URL сервера.
- **Доступ по WebDAV** — монтируйте свой диск по WebDAV с персональными паролями приложений
  (совместимо с RFC 4918 Class 2).
- **Серверное архивирование** — ставьте в очередь потоковые задачи ZIP и отслеживайте их на новой
  странице фоновых задач.
- **Загрузка папок** в веб-интерфейсе.
- **Облачные кредиты** — тарифицируемый исходящий трафик хранилища, оплачиваемый кредитами, с магазином кредитов.
- **Капча** для защиты входа и регистрации.
- Унифицированное управление API-ключами.

### Исправления
- Укреплён жизненный цикл удалённой загрузки (сбросы, восстановление и обработка сидов),
  а также различные исправления предпросмотра и загрузки.

> **Несовместимое изменение:** более строгие RESTful-маршруты API; публичные ссылки на загрузку перенесены с
> `/dl/:token` на `/r/:token`.

[Full release notes ↗](https://github.com/saltbo/zpan/releases/tag/v2.7.0)

## v2.6.0 — 2026-05-10 · Pro-лицензирование и магазин квот

### Возможности
- **Pro-лицензирование** — свяжите свой экземпляр с ZPan Cloud (QR + модальное окно сопряжения),
  права, проверенные через Ed25519, с фоновым обновлением, и ограничение доступа к Pro-функциям.
- **White-label брендинг** — собственный логотип, фавикон, текстовый логотип и скрытый футер.
- **Магазин квот** — коды погашения, ежемесячные квоты трафика, пакеты подписки и
  фиксированных квот, тарификация по объёму в разных валютах и превышение трафика.
- **Админ** — журналы аудита по изменяющим состояние действиям, объявления на сайте,
  регистрация по приглашениям и переработанная панель настроек и обзора.
- Предпросмотр файлов получил просмотрщик Microsoft Office, музыкальный плеер и
  очередь прогресса загрузки нескольких файлов.

### Исправления
- Раздел биллинга перенесён в административную панель; единицы квот, пользовательские квоты и аватары в админке.
- Фоновая синхронизация использования облачного трафика; контроль ежемесячного трафика на ссылках загрузки.

[Full release notes ↗](https://github.com/saltbo/zpan/releases/tag/v2.6.0)
