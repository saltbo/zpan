# v2.4 — Branding & Polish

Make ZPan feel like your own product. Customizable branding and visual polish.

## Major Features

### Branding
- Custom site name, description, and favicon
- Custom logo in the navigation bar
- User avatar upload (stored in S3)

### Visual Polish
- Dark mode
- Custom file access domain per storage backend (e.g. `img.mysite.com`)
- Clean URLs for image bed use: `img.mysite.com/abc123.png`

## User Scenarios

**Blogger with a personal brand:**
> I set my logo, site name "Yan's Images", and custom domain `img.yan.dev`. My image URLs look like they come from my own service, not a third-party tool.
