# v2.8 — Managed Service

Launch the hosted version of ZPan with payments and platform-exclusive features.

> These features are **managed version only** — they require server-side infrastructure. Self-hosted ZPan remains full-featured and free through v2.7.

## Major Features

### Payments & Plans
- Stripe subscription billing
- Free / Pro / Team tiers with defined limits
- Usage metering (storage, API calls, bandwidth)
- Self-service upgrade / downgrade
- Grace period — plan expires, files preserved (read-only) for 30 days

### Analytics Dashboard
- Storage usage trends over time
- Bandwidth consumption per day/week/month
- Popular files ranked by downloads / views
- Share link analytics (views, downloads, geographic distribution)
- API usage (call counts, error rates)

### Webhook Notifications
- Fire HTTP callbacks on events (file uploaded, deleted, share accessed, quota warning)
- Retry with exponential backoff
- Delivery history and debug logs

### Audit Log
- User activity log (uploads, downloads, deletes, share creation)
- Admin action log (user management, storage changes)
- Filterable by user, action type, date range
- CSV/JSON export for compliance

### Site Branding (white-label)
- Custom site logo (replaces ZPan wordmark in nav and auth pages)
- Custom favicon
- Stored under `_system/branding/` in the workspace's S3 bucket
- Managed-plan tier-gated. Self-hosted users keep the default ZPan brand.
- Future: custom brand colors and fonts (evaluated post-v2.8 based on Pro / Team demand)
