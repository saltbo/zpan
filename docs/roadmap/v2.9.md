# v2.9 — Managed Advanced

Advanced platform features for the managed version.

> These features are **managed version only**. Self-hosted ZPan remains full-featured and free.

## Major Features

### Content Moderation
- Upload scanning against moderation API (NSFW detection, etc.)
- Admin review queue for flagged files
- Auto-suspend accounts with repeated violations
- Configurable content policies

### Server-side Processing
- Archive extraction — upload a zip, extract contents to S3 via background worker
- Thumbnail generation — auto-generate image thumbnails
- File format conversion (e.g. HEIC to JPEG)

### Managed Custom Domains
- One-click custom domain — enter domain, platform provisions SSL and routing
- Guided DNS verification with CNAME instructions
- Vanity share URLs using your domain

> Self-hosted users already configure custom domains by pointing their own DNS (v2.4). This managed feature automates SSL and routing.
